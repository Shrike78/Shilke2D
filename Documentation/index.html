Core.Assets Assets namespace provides facilities to load different type of resources.
Core.Event Event objects are passed as parameters to event listeners when an event occurs.
Core.EventDispatcher EventDispatcher is the base class for all classes that dispatch events.
Core.IAnimatable The IAnimatable interface describes objects that are animated depending on the passed time.
Core.Juggler A Juggler animates objects implementing IAnimatable interface.
Core.Keymap KeyMap definition based on GLUT host keymap
Core.Shilke2D Shilke2D Moai version Shilke2D is the main class of Shilke2D lib.
Core.Timer Timer class is the equivalent of as3 Timer.
Core.TouchSensor Touch Sensor is an inner class used by Shilke2D to handle mouse and touch events.
Display.BaseQuad BaseQuad is a displayObj, base class for all the displayObject that can be rapresented by a quad (so Quad, Image, Movieclip and so on)

Is not a real 'renderable' because has no MOAIDeck binded to the main prop, it can be considered as a middle class that has no meeaning to be instantiated by itself.
Display.Button A simple button composed of an image and, optionally, text.
Display.DisplayObj The DisplayObject derives from EventDispatcher and is the base class for all objects that are rendered on the screen.
Display.DisplayObjContainer A DisplayObjectContainer represents a collection of display objects.
Display.DrawableObject A DrawableObject is a displayObj that implements MOAIScriptDeck, so that allows vectorial drawing.
Display.Image An Image is the Shilke2D equivalent of Flash's Bitmap class.
Display.MovieClip A MovieClip is a simple way to display an animation depicted by a list of textures.
Display.Quad A Quad represents a rectangle with a uniform color or a color gradient.
Display.Stage Stage is a particular DisplayObjContainer, root node of the displayList tree.
Display.TextField A TextField is a particolar displayObject used to displays text.
Texture.BigTextureAtlas BigTextureAtlas inherits TextureAtlas and allows to work with different atlas textures without caring about how textures are grouped together.
Texture.SubTexture A SubTexture represents a section of another texture.
Texture.Texture A texture stores the information that represents an image.
Texture.TextureAtlas A texture atlas is a collection of many smaller textures in one big image.
Texture.TextureManager TextureManager allows to mount a texture atlas as a logical resource path.
Texture.TexturePacker There are several ways to create a texture atlas.
include Shilke2D/include is the entry point for each project based on Shilke2D
