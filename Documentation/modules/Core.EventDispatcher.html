Module Core.EventDispatcher

EventDispatcher is the base class for all classes that dispatch events.

This is the Shilke2D version of the Flash class with the same name. Objects can communicate with each other through events.

Compared the the Flash event system, Shilke2D's event system was highly simplified. They are simply dispatched at the target. As in the conventional Flash classes, display objects inherit from EventDispatcher and can thus dispatch events.

It's possible to register a function or a method as event listenr of a specific event type for an object that derived from EventDispatcher

EventDispatcher:addEventListener (eventType, listenerFunc, listenerObj)
Register a function or a method as listener of a certain type of event. If nil listenerFunc is considered as a function

Parameters:

  • eventType the type of event for which the listener is registering
  • listenerFunc function registered as callback for the event
  • listenerObj if provided, listenerFunc is considered as a method of listenerObj.
