math.fequals (x1, x2, dist)
Checks if 2 numbers are equals "enough". Because for floating point numbers the equals operator raises problems, it's possible to use this function the check if the difference bewteen 2 number is as small as desired.

Parameters:

  • x1 first number
  • x2 second number
  • dist maximum distance allowed to consider the 2 number equals. default value is eps

Returns:

    bool
