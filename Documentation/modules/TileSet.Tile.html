Module TileSet.Tile

Tile is the base object for tilemap management.

A tile is always part of a tileset, a collection of tiles, and it's nothing more than a wrapper of a texture with some more properties.

It's possible to create custom tiles extending Tile class and registering new classes using TileManager.
