Module Utils.Color

Color class.

- Wraps r,g,b,a floating number. There is no real assumption about the range value, they can be also negative - the unpak_normalized() call is the only call that assumes values in the range of [0,255] and that returns a value divided by 255, so normalized in MOAI [0,1] color space
