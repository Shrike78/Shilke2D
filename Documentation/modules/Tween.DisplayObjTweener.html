Module Tween.DisplayObjTweener

DisplayObjTweener is a tween helper to animate displayObjs.

It's a set of wrapper function over Tween.ease() that allows to easily animate displayObjs properties (position, scale, rotation, color, alpha). Default transition for each transiformation is Transition.LINEAR
