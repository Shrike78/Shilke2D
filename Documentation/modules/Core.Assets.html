Module Core.Assets

Assets namespace provides facilities to load different type of resources.

Fields

__defaultCacheTexture by default texture are cached.


Functions

getSound (fileName)
Load a sound.

Parameters:

  • fileName the name of the sound to load, relative to the working dir or absolute (starting with /)

Returns:

  1. a sound object if fileName is a valid path, else nil
  2. an error message if fileName is not a valid path
getXml (fileName)
Load an Xml.

Parameters:

  • fileName the name of the xml to load, relative to the working dir or absolute (starting with /)

Returns:

  1. an XmlNode if fileName is a valid path, else nil
  2. an error message if fileName is not a valid path
getRawImage (fileName)
Load a raw image.

Parameters:

  • fileName the name of the raw image to load, relative to the working dir or absolute (starting with /)

Returns:

  1. a MOAIImage if fileName is a valid path, else nil
  2. an error message if fileName is not a valid path
getTexture (fileName, useCache)
Load a Texture.

Parameters:

  • fileName the name of the texture to load, relative to the working dir or absolute (starting with /)
  • useCache override the default beahviour for caching logic

Returns:

  1. a Texture if fileName is a valid path, else nil
  2. an error message if fileName is not a valid path
clearTextureCache (texture)
Clear the texture cache.

Parameters:

  • texture can be a Texture or a string (the name of the asset on which the Texture was created). If no param is provided it clears the whole textureCache
