Module Display.BaseQuad

BaseQuad is a displayObj, base class for all the displayObject that can be rapresented by a quad (so Quad, Image, Movieclip and so on)

Is not a real 'renderable' because has no MOAIDeck binded to the main prop, it can be considered as a middle class that has no meeaning to be instantiated by itself.

BaseQuad allows different pivotMode. Default PivotMode is PivotMode.CENTER
