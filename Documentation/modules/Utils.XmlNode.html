Module Utils.XmlNode

XmlNode is an helper class to work with xmls.

A XmlNode object has a name, a value, attributes, children nodes and a father. An XmlNode without a father is a root node. It's possible to convert product of XmlParser into an XmlNode. It's also possible to add / remove children and attributes from a node.
