Module Core.Timer

Timer class is the equivalent of as3 Timer.

Each timer can be scheduled to run multiple times, even infinite. It's possible to register as listener for timer events. A Timer can be started, stopped and paused

Functions

init (delay, repeatCount) Timer initialization.
start () Starts the timer, if it is not already running.
stop () Stops the timer, if it is running.
reset () Stops the timer, if it is running.
