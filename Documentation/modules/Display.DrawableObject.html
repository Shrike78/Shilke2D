Module Display.DrawableObject

A DrawableObject is a displayObj that implements MOAIScriptDeck, so that allows vectorial drawing.

It's an abstract class, in fact it doesn't implements the getRect() method, and is the base class of all the object that make vectorial drawing.

Concrete objects that inherits from this class need to implements the DisplayObj:getRect() method and the new DrawableObject:_innerDraw()
