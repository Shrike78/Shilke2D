Module Core.TouchSensor

Touch Sensor is an inner class used by Shilke2D to handle mouse and touch events.

Initializes the mouse/touch sensors and set the proper callbacks that can be redefined by user.

Mouse left click and move are redirected as touch events with always the same id.

Touch phases are {BEGAN, MOVING, ENDED, CANCELLED}

onEvent (eventType, idx, x, y, tapCount)
Inner event handling method to which touch and mouse events are redirected

Parameters:

  • eventType MOAITouchSensor event type on which redirect touch phase
  • idx the id of the current touch event in progress. on desktop id is always -1
  • x logical stage x coordinate of the touch event
  • y logical stage y coordinate of the touch event
  • tapCount for multiTap touch event the idx value doesn't change but the tapCount increase each tap
